Список команд git в Terminal VSC.

  • git status - перевірка статусу передачі даних до репозиторію Github.
  • git add "назва файлу"- додання конкретних нових даних/файлів до репозиторію Github.
  • git add . - додання всіх нових даних/файлів до репозиторію Github.
  • git commit -m "текс на англійскій мові" - коментарі щодо інформації, яка передається до репозиторію Github.
  • git push - додання нових даних/файлів до репозиторію Github.
  • git push origin "навзва гілки"- додання нових даних/файлів до репозиторію Github.
  • git push origin HEAD- додання нових даних/файлів до репозиторію Github відповідно до останього коміту.
  • git checkout -b "назва нової гілки" - створення нової гілки в репозиторії Github.
  • git checkout назва папки - перехід між гілками репозиторію Github
  • git branch - перевірка наявних гілок в репозиторії Github

Порядок дій в Terminal VSC.

  • git status
  • git add "назва файлу"
  • git add .
  • git commit -m "текс на англійскій мові"
  • git push
  • git push origin "навзва гілки"
  • git push origin HEAD
